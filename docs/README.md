## Histórico arquivado

A pasta [`_legado/`](_legado/) preserva a documentação da fase anterior do projeto (chat mascarado / orquestração multi-agente, de abril–maio de 2026), antes do pivô para o canvas. É mantida como **trilha histórica** — não descreve o produto atual, mas registra decisões, planos e relatórios daquela etapa.

Os padrões de design, prompts-base e o template de contexto **não** vivem aqui — são do repositório [Felixo System Design](https://github.com/Felipe-Alcantara/Felixo-System-Design), referenciado pela pasta `Padrão de qualidade - Felixo System Design/` (quando presente) ou diretamente pela fonte.
